# JQueryMobile App

## Aplicacion simple desarrollada en JQuery Mobile para la seleccion de vuelos a distintos paises.

Muestra la seleccion de paises, provincias y localidades consumidas de una base de datos.
Se puede incorporar como modulo de seleccion en aplicaciones mas robustas.
Ademas luego muestra vuelos, tambien cargados en una base de datos, para el pais y la provincia seleccionada
